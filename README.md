# iocage-plugin-i2p
Artifact files for I2P iocage plugin

# Manual installation
```
git clone https://github.com/dbzoo/iocage-plugin-i2p.git
cd iocage-plugin-i2p
iocage fetch -P i2p.json --name i2p
```
